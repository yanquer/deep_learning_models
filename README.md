# deep_learning_models

## 前言

个人看来，由于现在科技的局限性，机器真正认识的只有二进制0与1，
所以深度学习的本质在于寻找一个适用性强的函数。

另一方面，由于现在科技发展迅速，机器有能力处理大批量数据，
某些深度函数的结合无法有效被证实，所以很难说出为什么这个函数/方法是有效的，
从这方面看，深度学习也具有了一些哲学与玄学的色彩。
